## What was tried

Buck's call on this band was blunt: the old explorations drifted, the
references are the standard. So this round starts over from the reference
(the Field Studies Flora field guide — studied frame by frame from the ref
video, not from memory) and traces its actual moves:

- **var-1 — catalog grid, faithful.** Two-column grid split by hairlines;
  each entry is roman numeral + tracked small-caps label + italic serif
  sub, offset on one baseline; typewriter key/value metadata in two
  mini-columns under the image; small serif body below. Six plates on the
  property (facade, courtyard, lobby, sign, arcade, street) — facts from
  06 knowledge.
- **var-2 — chapter page.** The ref's "V. FALL" spread: big letterspaced
  typewriter numeral + title, INDEX: with bracketed entries, dated
  small-caps sidebar row, italic serif intro at center, full-width image.
- **var-3 — typeface cycle.** Buck named typefaces as the next axis, so
  the same grid re-set with Cormorant body against the typewriter
  metadata. Further faces are cheap to add — the grid takes `bodyFace`.

## Calls made (agent, review-level)

- **Courier Prime added as `font-typewriter`** — the ref's metadata blocks
  are set in a typewriter face and the project had no mono. This is the
  single biggest "je ne sais quoi" carrier: the offset typewriter
  key/value block against serif body.
- Numeral style rendered as the ref shows it (roman + period, typewriter)
  but treated as **provisional** — 15 numbering system owns the brand-wide
  call, per the doc's cross-tie note.
- Old explorations were not consulted for composition — doc's negative:
  "Don't build on them — build from the refs again."
