## What was tried

The exploration axis is the layering system itself — which levels exist
and how many. This round proposes a named level system and tests it at
two densities on room plates:

- **M1 · plate/register** — typewriter 10px, wide-tracked upper; locates
  the composition in a sequence (plate no. left, class code right)
- **M2 · metadata** — typewriter 10.5px key/value; the facts layer
- **M3 · annotation** — serif italic 12px; the one human aside, keyed to
  a real fact (James Dean Room, 223)
- **M4 · provenance** — 9px tracked upper at lowest contrast; sourcing
  (the honest "a king room shown" label lives here)

**var-1** runs all four on a Queen Historic plate. **var-2** strips to
M2 + M4 on 2 Queen Historic — testing how few levels still read as
layered rather than just captioned.

## Calls made (agent, review-level)

- Every string is sourced (06 knowledge, lib/data, the media doc's honest
  label) — the band's one hard kill is random/meaningless microtext, so
  meaning was treated as a build constraint, not a styling note.
- Typewriter voice shared with 09 — the two bands develop against the
  same ref crops per the doc's cross-tie.
- M3 capped at one per composition; more asides would flatten the levels.
