# Stacked Typographic Menu Layout

## Description

This layout demonstrates a dense vertical stacking of information using varied typographic weights and styles to create a complex information hierarchy. The arrangement utilizes centered serif titling with high-contrast stroke widths, though the density of the all-caps sans-serif subtext creates a cluttered visual field. Squiggly line dividers and wide-tracked spacing are used to separate the numerous content blocks, resulting in a busy, maximalist editorial aesthetic.
