# Oversized Script on Canvas Texture

## Description

This reference features a high-resolution canvas weave texture with visible gesso priming and subtle horizontal seam lines, creating a tactile, physical backdrop. The typography utilizes an oversized, fluid script that bleeds off the edges, demonstrating a 'full-bleed' compositional technique where the letterforms act as both content and structural framing. The contrast between the organic, woven fabric substrate and the sharp, cream-colored vector-style lettering provides a sophisticated interplay of digital and analog aesthetics.
