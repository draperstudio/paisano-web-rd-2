# 25 wordmark redraw — raw feedback

Buck's board comment on this band, pulled verbatim from the Figma board (Explore page), board pass 7.12.26.

I came across a reference using oversized script typography and immediately liked it. My first thought was, "We should do something like this for the preloader." It was never about making this the logo—it was about using it as a large typographic moment on the website.
When I had AI start exploring that direction, it naturally wasn't using our logo. Different models generated different script typefaces, and seeing all of those side by side actually accentuated something I already felt but hadn't really acknowledged: I don't like our current logo very much.
Those script explorations weren't perfect, and they couldn't simply become the logo. The current logo comes directly from the existing neon sign on the building, and that sign is not changing. Because of that, we can't completely redesign the logo into one of these elegant scripts. If we departed that far from the sign, the relationship between the identity and the physical building would break, and I don't think that's the right move.
The preloader is different because it is intentionally not the logo. It's clearly just a graphic treatment. That gives us much more freedom to use a beautiful script without creating confusion. If anything, we'd push it even further so it's obvious this is an expressive typographic moment rather than the brand mark itself.
The logo, however, sits in a different position. I don't think it should become one of these scripts, but I also don't think it should stay exactly as it is.
What the AI explorations did was expose the gap. They made me realize there are qualities in our logo that feel off. It feels a little too digital, a little too glossy, a little too manufactured—like somebody drew it in Photoshop twenty years ago instead of it being a naturally drawn piece of lettering.
The opportunity isn't to replace the logo. It's to refine it.
The neon sign has characteristics that make it recognizable—its overall structure, proportions, and little details like the tails on certain letters. Those are what connect it back to the building, and we should preserve that relationship. But I think there's room to redraw the logo so it moves slightly toward the qualities these script explorations have. It shouldn't become the script, but it should become more natural, more fluid, and better drawn while still clearly feeling like the same sign.
In hindsight, the AI explorations accomplished two different things. First, they revealed a strong direction for the website through these oversized script treatments that can be used in places like the preloader. Second, they exposed weaknesses in the current logo and suggested a direction for refining it. Those are two separate decisions, and I don't think they should be conflated. The website can be much more expressive with typography because those treatments are not functioning as the logo, while the logo itself should evolve much more conservatively so it remains authentic to the historic neon sign.
