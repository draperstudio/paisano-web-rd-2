# Round two — v0, 7.12.26

Nothing removed; this expression had no built variants before this
round (round one was a record-only pass), so var-1 is its first.

## What the ref actually is

An asymmetrical grid with centered typography: a full-width brand-color
band across the top carrying the inverted mark; below it a two-column
field — the room photo large on one side, and on the other a centered
type stack (class numeral, room name, dedication line, then the fact
rows), with the whole grid held inside a page margin.

## Built

- var-1 — the room detail frame on the ref's anatomy: maroon band with
  the inverted current mark; Courtyard King photo left; centered stack
  right (numeral II, name, description, bed/sqft/rate rows separated
  by hairlines).

Also in this round: CurrentMark gained an `inverted` prop (CSS filter
to cream) so the dark PNG mark can sit on brand-color bands — the ref
does exactly this with its logo on red.
