# 07 script mixed in — expressions doc

**Version:** V1 · 7.12.26 · created from the board pass (Buck's board comment, worked by AI — Buck QAs)
**Type:** composition concept — an explanatory section pattern built on one typographic move (script face mixed inside normal running text).

## What it is

A section where the brand explains something, with a script face mixed into normal text mid-line. Round one produced a couple of explorations. The existing ones miss on the details — the all-caps kerned body text and the particular script both fail — but the move itself is strongly liked. The reference also carries a texture that ties to the texture asset (28).

## What Buck likes

"I really like this one of mixing in fucking script face within normal text... this idea of having a section where we explain something — I don't know what we would explain in this type of section but I'd like it a lot. I like it for sure." And the texture: "maybe we make some texture like this and this is where we bring similar texture back in to what we did if we do the preloader thing at the top."

## Control for (locked)

- **Type (voice-switching):** script face mixed within normal running text, mid-line register changes — the defining move.
- **Subject:** an explanatory section — the brand talking, a note-from-the-house register. (What it explains is a content-concept question, open.)
- **Texture tie:** if the preloader texture (01 + 28) happens, this section is where the texture comes back — a deliberate echo across the page.

## Leave open (variation space)

- **Typefaces — both sides of the mix:** "there's a lot of fine nuanced je ne sais quoi that needs to be figured out as far as what the typefaces we use are and the kerning and the sizing and shit." The exploration's script is dead; the replacement is open.
- **Casing:** genuinely unresolved — all-caps-with-kerning is dead for the normal text, but "I also don't think sentence case or title case would work. I don't know. We're just gonna have to try it with different typefaces." Cycle casing × typeface together.
- What the section explains — content concept, later pass.
- **Grid / Space / Color / Zones.**

## Negative

- All caps with wide kerning for the normal text: "I don't like the all caps with quite a bit of kerning."
- The exploration's script type: "I also don't really love that script type."
