# Split-Field Card with Footer Alignment

## Description

This layout utilizes a vertical split with a full-bleed image occupying the top half and a clean white field below for typography. The primary interest lies in the footer arrangement, where serif text is anchored to the bottom-left and a numerical indicator is perfectly aligned to the bottom-right, creating a balanced horizontal axis. This structural approach provides a clear hierarchy by separating visual storytelling from functional metadata through distinct spatial zones.
