# Stacked Card Layout with Logo Overlay

## Description

This layout utilizes a precise graphic system featuring a thin, uniform negative space border around a centered image, with a large serif logo overlaid directly onto the visual. The typography is strictly center-aligned, utilizing a vertical stacking method that places primary copy immediately below the image frame and secondary details further down the card's footer. The composition balances high-contrast white space with structured text blocks to create a sophisticated, editorial hierarchy.
