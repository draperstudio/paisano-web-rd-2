# Vertical Rule

**Code:** HP-GR-004

**Section:** graphics

**Type:** Rule

A tall column divider in near-black ink with a subtle inner shadow, so it reads as an engraved seam between two columns.

## Usage

**When to Use**
- To divide a two-column layout, such as a text column beside a data panel

**Do Not**
- Use it horizontally
- Place more than one per spread
- Increase its weight

## Specifications

Color: Ink #01090F
Weight: Hairline with a light inner shadow
Height: Full column height
Placement: Centered between two balanced columns
File: SVG
