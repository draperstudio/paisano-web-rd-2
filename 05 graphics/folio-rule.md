# Folio Rule

**Code:** HP-GR-002

**Section:** graphics

**Type:** Rule

The page-number marker. A numeral, a hairline, and a diamond terminal that runs from the page number toward the outer edge. It sits in the top-right of the running header, opposite the section label.

## Usage

**When to Use**
- In the top-right header of multi-page documents and decks, carrying the current page number

**Do Not**
- Use it as a divider inside the body
- Show it without a page number attached

## Specifications

Color: Oxblood #42101B
Position: Top-right running header
Weight: Hairline, matching the top rule
Numeral: Body serif, oldstyle figures
File: SVG
