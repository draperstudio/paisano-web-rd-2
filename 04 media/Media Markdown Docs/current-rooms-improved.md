# current-rooms-improved/

## Description

First outputs of the doctored-but-still-accurate-to-current property photos of the guest rooms as they are today (Buck pasted these 7.12.26; more coming from the Paisano-Room-Photo-Work flow). Doctored means improved photography — cleaned, corrected, better shot — not imagined renovation: everything shown exists at the property now.

## Usage rule (settled — see `01 settled/settled.md` → Imagery)

Any expression showing a current room uses these (or other current photography) — never the renders. Same rule for courtyard, lobby, facade: current is the standard state, show the actual property. Renders only where the piece is explicitly about the rebuild / next steps / the future.
